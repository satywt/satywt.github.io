# 城市招牌字计划
现代城市中充斥着太多快速制作而显得非常廉价、毫无设计感的招牌字，很多时候甚至没有手写招牌来得美观、真挚。

因此，我决定每周为一家小店，重新设计招牌字。
由于并非店主委托，作为个人项目，设计会有（非常）严重的个人审美倾向。
